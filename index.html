🚀 New-API Helm Chart Repository

这是 New-API 的官方 Helm Chart 仓库，提供高可用的 OpenAI API 网关服务部署解决方案。

Kubernetes Helm Chart High Availability Load Balancer Auto Scaling

📦 安装方式

# 添加 Helm 仓库 helm repo add new-api https://<your-username>.github.io/<repository-name> # 更新仓库 helm repo update # 安装 Chart helm install my-new-api new-api/new-api

🎯 主要特性

  • 智能负载均衡 - 根据启用组件自动配置后端选择
  • 自动外部访问 - Ingress未启用时自动转换为NodePort服务
  • 高可用架构 - 支持Master-Slave部署模式
  • 自动扩缩容 - 集成HPA水平Pod自动扩缩容
  • 数据库支持 - 内置MySQL和Redis高可用配置
  • 配置灵活 - 丰富的values.yaml配置选项

🔧 快速配置

# 基础部署 helm install new-api new-api/new-api # 高可用部署 helm install new-api new-api/new-api \ --set app.master.enabled=true \ --set app.slave.enabled=true \ --set app.slave.replicaCount=3 \ --set autoscaling.enabled=true # 自定义NodePort helm install new-api new-api/new-api \ --set service.nodePort=30080 \ --set ingress.enabled=false
